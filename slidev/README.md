# Slidev Sunumu

## Kurulum

```bash
cd slidev
npm install
```

## Kullanım

### Geliştirme Modu

```bash
npm run dev
```

Tarayıcıda `http://localhost:3030` adresinde sunum açılacak.

### Export

```bash
# PDF olarak export
npm run export

# HTML olarak build
npm run build
```

## Özellikler

- ✅ Sağlı sollu kod karşılaştırması
- ✅ Syntax highlighting
- ✅ Line highlighting (vurgulama)
- ✅ Koyu tema
- ✅ Manuel kontrol (klavye ile ilerleme)
- ✅ PDF export desteği

## Klavye Kısayolları

- `Space` veya `→`: Sonraki slide
- `←`: Önceki slide
- `f`: Fullscreen
- `o`: Overview
- `g`: Go to slide
